# Dot Env

Библиотека для быстрой и не сложной работы с переменными окружения в Python. Работать с библиотекой так же просто как и Pydantic Settings. Но моя библиотека не требует установки дополнительных библиотек а так же имеет доболнительные возможности.

```python
from dot_env import DotEnvSettings

class Settings(DotEnvSettings):
    TEST_STR: str
    SPECIAL_INT: int
    DEBUG: bool
    
settings = Settings('.env.test')
```

В данном примере при инициализации класса, открывается файл `.env.test` и загружаются переменные окружения в заданные атрибуты класса с указанным типом данных.

## Дополнительные возможности

Библиотека поддерживает автоматическое разделение списков и обработку их элементов к указанным типам данных. Изначально разделителем для всех списков является `|`, но вы можете изменить её в момент инициализации класса указав `separator=','`. Базого все элементы списка являются строкой. Так же рекомендуется в типах данных указывать кортеж.

Если вам нужно преобразование элементов кортежа импортируйте дополнительные типы:

```python
from dot_env import DotEnvSettings, TupleInt, TupleFloat

class Settings(DotEnvSettings):
    IDS: TupleInt       # Все элементы будут приобразованны в int
    PRICES: TupleFloat  # Все элементы преобразаются во float

settings = Settings('.env.test')
```

## Исключения

- В случае, если файл по пути не найден или не может быть прочитан, будет вызвано исключение `EnvFileNotExists`. 
- Если переменная окружения не найдена, будет вызвано исключение `LoadTokenException`.
- Если не удалось преобразовать значение переменной окружения в указанный тип данных, будет вызвано исключение `ValueTypeException`.
- Если в файле будут найдены другие ключи вызовится исколючение `AnotherValueFound`
- Если тип переменной не поддерживается, вызовится исколючение `UnsupportedTypeException`

## Передача параметров

Рекомендуется передавать параметры в формате kwargs:

```python
settings = Settings(
    file_path='.env.test',
    separator=','
)
```

Или можно задать параметр при описании класса:

```python
class Settings(DotEnvSettings):
    __file_path = '.env.test',
    __separator=','
    __encoding='utf-6'
```
